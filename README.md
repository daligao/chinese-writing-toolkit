## What is this?

A browser-based Chinese writing practice tool built for **English speakers learning Mandarin**. Covers the most essential letter/text types used in HSK writing exams, business Chinese, and everyday communication.

No install. No login. Single HTML file — just open and start writing.

The **full version** (11 types including 求职信 Cover Letter and 自我介绍 Self-introduction) is hosted at [ordinarymantrying.com](https://ordinarymantrying.com/tools/chinese-writing-toolkit.html). This repo contains the public 9-type version.

## What's inside

### ✉️ 9 Applied Writing Types 应用文写作 (this repo)

| Type | Chinese | English |
|---|---|---|
| Invitation | 邀请信 | Invitation Letter |
| Thank-you | 感谢信 | Thank-you Letter |
| Advice | 建议信 | Letter of Advice |
| Apology | 道歉信 | Letter of Apology |
| Application | 申请信 | Application Letter |
| Notice | 通知 | Notice / Announcement |
| Speech | 演讲稿 | Speech |
| Inquiry | 询问信 | Letter of Inquiry |
| Complaint | 投诉信 | Complaint Letter |

**Full version adds:** 求职信 (Cover Letter) + 自我介绍 (Self-introduction) → [ordinarymantrying.com](https://ordinarymantrying.com/tools/chinese-writing-toolkit.html)

Each type includes:
- **Structure guide** — what goes in each paragraph
- **Sentence banks** — Chinese sentences with English translations, tap 🔊 to hear them
- **Fill-in template** — plug in your details and go
- **Model essay** — full Chinese essay with hidden English translation

### 🛠 Word Upgrade Toolbox 提分工具箱
- 25 word upgrades: swap plain words (很好, 但是, 我认为) for high-level equivalents
- 7 connector groups (adding, contrast, cause-effect, example, sequence, summary, condition)
- **13 advanced grammar patterns** with examples and audio
- 10 formal phrases (套语) for professional letters
- 8 universal opening sentences
- 7 universal closing sentences

### 📝 Practice Pad 练习本
- Write Chinese directly in the browser
- Auto character count
- Save draft locally
- Self-check bilingual checklist (Chinese + English) before submitting
