# Chinese Writing Toolkit · 中文写作练习

> Write better Chinese — sentence banks, templates, and model essays for English speakers.

**[→ Try it live](https://ordinarymantrying.com/tools/chinese-writing-toolkit.html)** &nbsp;·&nbsp; **[GitHub Pages](https://daligao.github.io/chinese-writing-toolkit/chinese-writing-toolkit.html)**

---

## What is this?

A browser-based Chinese writing practice tool built for **English speakers learning Mandarin**. Covers the 8 most essential letter/text types used in HSK writing exams, business Chinese, and everyday communication.

No install. No login. Single HTML file — just open and start writing.

---

## What's inside

### ✉️ 8 Applied Writing Types 应用文写作

| Type | Chinese | English |
|---|---|---|
| Invitation | 邀请信 | Invitation Letter |
| Thank-you | 感谢信 | Thank-you Letter |
| Advice | 建议信 | Letter of Advice |
| Apology | 道歉信 | Letter of Apology |
| Application | 申请信 | Application Letter |
| Notice | 通知 | Notice / Announcement |
| Speech | 演讲稿 | Speech |
| Inquiry | 询问信 | Letter of Inquiry |

Each type includes:
- **Structure guide** — what goes in each paragraph
- **Sentence banks** — Chinese sentences with English translations, tap 🔊 to hear them
- **Fill-in template** — plug in your details and go
- **Model essay** — full Chinese essay with hidden English translation

### 🛠 Word Upgrade Toolbox 提分工具箱
- Swap plain words (很好, 但是, 我认为) for high-level Chinese equivalents
- Connectors by category (contrast, cause-effect, summary...)
- 8 advanced grammar patterns that impress examiners
- Universal opening and closing sentences

### 📝 Practice Pad 练习本
- Write Chinese directly in the browser
- Auto character count
- Save draft locally
- Self-check checklist before submitting
